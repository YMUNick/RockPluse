# Rock Pulse

**Rock Pulse** is a single-page web application that tracks real-time Instagram rock & band trends and generates band-frontman-style social posts with AI.

## Features

- Real-time Instagram rock & band trend tracking dashboard
- AI-powered content engine for generating rock/band-style social media posts
- Dark brutalist UI design with JetBrains Mono & DM Sans typography
- Fully self-contained single HTML file — no build step required

## Tech Stack

- Pure HTML / CSS / JavaScript (no framework)
- Google Fonts (JetBrains Mono, DM Sans)
- Anthropic Claude API for AI content generation

## Usage

Open `rock-pulse-en.html` directly in any modern browser. No server or installation needed.
